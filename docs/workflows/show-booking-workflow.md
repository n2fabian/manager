# Show Booking Workflow

## Overview
This workflow guides you through the process of booking and executing a successful show.

---

## Phase 1: Research & Outreach

### 1.1 Identify Target Venues
- [ ] Research venues in target city
- [ ] Check venue social media for genre fit
- [ ] Note capacity and typical draw
- [ ] Find booking contact information
- [ ] Add to `/wiki/contacts/venues.md`

### 1.2 Prepare Materials
- [ ] Update tech rider for specific band
- [ ] Prepare EPK/press kit
- [ ] Have recent live videos ready
- [ ] Have streaming links ready

### 1.3 Initial Contact
- [ ] Use template: `/docs/templates/venue-email-template.md`
- [ ] Customize for specific venue
- [ ] Include links to music/video
- [ ] Attach tech rider
- [ ] Propose 2-3 date options
- [ ] Log outreach in CRM

**Wait 1-2 weeks for response**

---

## Phase 2: Negotiation & Confirmation

### 2.1 Follow Up (If No Response)
- [ ] Send follow-up email after 1-2 weeks
- [ ] Keep it brief and friendly
- [ ] Update CRM with follow-up date

### 2.2 Negotiate Terms (If Response Received)
- [ ] Discuss date
- [ ] Confirm set time and length
- [ ] Discuss payment (guarantee/door split/percentage)
- [ ] Clarify load-in and soundcheck times
- [ ] Confirm backline/technical requirements
- [ ] Discuss promotion responsibilities

### 2.3 Confirm Booking
- [ ] Send confirmation email with all details
- [ ] Add to calendar/show tracker
- [ ] Update venue relationship status in CRM
- [ ] Create task list for show preparation

---

## Phase 3: Pre-Show Preparation

### 3.1 Technical (2-4 weeks before)
- [ ] Send updated tech rider if needed
- [ ] Confirm load-in time
- [ ] Confirm soundcheck time
- [ ] Arrange transportation/parking
- [ ] Pack equipment checklist

### 3.2 Promotion (2-4 weeks before)
- [ ] Get event page/poster from venue
- [ ] Share on all social media platforms
- [ ] Create Facebook event (if not provided)
- [ ] Add to website calendar
- [ ] Email mailing list
- [ ] Create countdown posts
- [ ] Coordinate cross-promotion with venue
- [ ] Post multiple times leading up to show

### 3.3 Logistics (1 week before)
- [ ] Confirm all details with venue
- [ ] Check weather (for load-in planning)
- [ ] Prepare merch for sale
- [ ] Plan setlist
- [ ] Arrange guest list if applicable
- [ ] Confirm payment terms

---

## Phase 4: Show Day

### 4.1 Load-in & Setup
- [ ] Arrive at confirmed load-in time
- [ ] Unload equipment
- [ ] Set up according to stage plot
- [ ] Be respectful of venue and other bands

### 4.2 Soundcheck
- [ ] Arrive on time for soundcheck
- [ ] Check all instruments and vocals
- [ ] Communicate clearly with sound engineer
- [ ] Get monitor mixes dialed in
- [ ] Run through problem spots if time allows

### 4.3 Before Show
- [ ] Set up merch table
- [ ] Socialize with other bands/promoters
- [ ] Stay hydrated
- [ ] Final gear check
- [ ] Mental preparation

- [ ] Hang around and meet fans
- [ ] Sell merch

---

## Phase 5: Post-Show Follow-up

### 5.1 Immediate (Day After)
- [ ] Post thank you on social media
- [ ] Tag venue and other bands
- [ ] Share any photos/videos
- [ ] Thank fans who attended

### 5.2 Week After
- [ ] Send thank you email to venue
- [ ] Express interest in returning
- [ ] Update CRM with show notes
- [ ] Record payment in financial tracker
- [ ] Record any expenses

### 5.3 Long-term
- [ ] Add show to band history
- [ ] Save any professional photos/videos
- [ ] Add venue to "will work with again" list
- [ ] Plan next contact (3-6 months)

---

## Troubleshooting

### No Response to Initial Outreach
- Follow up once after 2 weeks
- If still no response, move on (they're busy, not personal)
- Try again in 3-6 months

### Low Turnout
- Analyze promotion efforts
- Consider partnering with local bands next time
- Don't blame venue - focus on building draw
- Still deliver great performance

### Technical Issues
- Stay calm and professional
- Work with sound engineer
- Adapt setlist if needed
- Don't complain to audience

### Payment Issues
- Have agreement in writing beforehand
- Be professional but firm
- Document everything
- Consider small claims if necessary (last resort)

## Success Metrics

Track these for each show:
- Attendance (estimated)
- Door/guarantee received
- Merch sold
- New email signups
- Social media engagement
- Venue relationship quality
- Would you play here again?
