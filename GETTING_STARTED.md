## Step 4: Create Your First Venue Outreach

### Research Phase
1. [ ] Pick 3-5 venues you want to play
2. [ ] Find their booking contact (usually on website or social media)
3. [ ] Check their calendar to see what kind of bands they book
4. [ ] Note their capacity and typical crowd
5. [ ] Add each venue to `/wiki/contacts/venues.md`

### Outreach Phase
1. [ ] Copy the email template
2. [ ] Customize for the specific band and venue
3. [ ] Attach the tech rider (PDF version)
4. [ ] Include links to your best music and live videos
5. [ ] Send it!
6. [ ] Track it in `/tasks/current-tasks.md`

### Follow-up
1. [ ] Set reminder for 1-2 weeks
2. [ ] Send brief follow-up if no response
3. [ ] Update CRM with status

## Tips for Success

1. **Start Small**: Don't try to fill everything out at once. Start with what you need for the next show.

2. **Build Habits**: Spend 15 minutes daily updating this system. It becomes automatic.

3. **Use the Workflows**: Follow `/docs/workflows/show-booking-workflow.md` for your first booking.

4. **Track Everything**: The more data you have, the better decisions you can make.

5. **Stay Organized**: This system only works if you use it consistently.

6. **Iterate**: Adjust templates and workflows to match your style and needs.

7. **Have Fun**: This is supposed to make your life easier, not harder!

## Common Questions

### "Do I need to fill out everything right away?"
Nope! Start with the basics:
- Band contact info
- One good email template
- One solid tech rider
- Current merch inventory

Everything else can be added as you go.

### "What if I don't have much to track yet?"
Perfect! This system grows with you. Start tracking now and you'll have valuable data as you build.

### "How do I use this with GitHub Copilot?"
The whole repository is designed to work with AI agents. When you open it in VS Code with Copilot:
- Ask questions about any file
- Request help generating emails or documents
- Get suggestions for content
- Automate repetitive tasks

Example prompts:
- "Generate a venue outreach email for Near's 2 Far to play at [Venue Name]"
- "Create a financial report for last month"
- "What venues should I contact for shows in [City]?"

### "Can I modify the templates?"
Absolutely! These are starting points. Customize everything to match your style and needs.
